◆ accrued()

Real accrued ( Date  settlement) const
private

accrued interest used internally, where includeToday = false

same as Bond::accruedAmount() but with enable early payments true. Forces accrued to be calculated in a consistent way for future put/ call dates, which can be problematic in lattice engines when option dates are also coupon dates.
