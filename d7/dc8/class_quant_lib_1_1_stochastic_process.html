Detailed Description

multi-dimensional stochastic process class.

This class describes a stochastic process governed by

\[ d\mathrm{x}_t = \mu(t, x_t)\mathrm{d}t + \sigma(t, \mathrm{x}_t) \cdot d\mathrm{W}_t. \]
