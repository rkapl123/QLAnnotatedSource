375  /*
376  Several (very different) usages make the spez non trivial
377  The final goal is to obtain a sequence generator of the factor
378  samples, several routes are possible depending on the algorithms:
379 
380  1.- URNG -> Sequence Gen -> CopulaInversion
381  e.g.: CopulaInversion(RandomSequenceGenerator<MersenneTwisterRNG>)
382  2.- PseudoRSG ------------> CopulaInversion
383  e.g.: CopulaInversion(SobolRSG)
384  3.- URNG -> SpecificMapping -> Sequence Gen (bypasses the copula
385  for performance)
386  e.g.: RandomSequenceGenerator<BoxMullerGaussianRng<
387  MersenneTwisterRNG> >
388 
389  Notice that the order the three algorithms involved (uniform gen,
390  sequence construction, distribution mapping) is not always the same.
391  (in fact there could be some other ways to generate but these are
392  the ones in the library now.)
393  Difficulties arise when wanting to use situation 3.- whith a generic
394  RNG, leaving it unspecified
395 
396  Derived classes might specialize (on the copula
397  type) to another type of generator if a more efficient algorithm
398  that the distribution inversion is available; rewritig then the
399  nextSequence method for a particular copula implementation.
400  Some combinations of generators might make no sense, while it
401  could be possible to block template classes corresponding to those
402  cases its not done (yet?) (e.g. a BoxMuller under a TCopula.)
403  Dimensionality coherence (between the generator and the copula)
404  should have been checked by the client code.
405  In multithread usage the sequence generator is expect to be already
406  in position.
407  To sample the latent variable itself users should call
408  LatentModel::latentVarValue with these samples.
409  */

800  urng_(seed) {

802  const std::vector<Real>& varF = copula.varianceFactors();
803  for(Size i=0; i<varF.size(); i++)// ...use back inserter lambda
804  trng_.push_back(
805  PolarStudentTRng<urng_type>(2./(1.-varF[i]*varF[i]), urng_));

808  Size i=0;
809  for(; i<trng_.size(); i++)//systemic samples plus one idiosyncratic
810  sequence_.value[i] = trng_[i].next().value;
811  for(; i<sequence_.value.size(); i++)//rest of idiosyncratic samples
812  sequence_.value[i] = trng_.back().next().value;
813  return sequence_;
814  }
