97  /* This function is just syntax since the only thing it does is calling
98  to integrate<RetType> which has to exist for the type returned by the
99  function. So theres one redundant call but there should not be any extra
100  cost... up to the compiler. It can not be templated all the way since
101  the integration entries functions can not be templates.
102  Most times integrands will return a scalar or vector but could be a
103  matrix too. Also vectors might be returned as vector or Disposable
104  wrapped (which is preferred and I have removed the plain vector
105  version).
106  */
107  template<class RetType_T>
108  RetType_T operator()(const ext::function<RetType_T (
109  const std::vector<Real>& arg)>& f) const
110  {
111  return integrate<RetType_T>(f);
112  }
