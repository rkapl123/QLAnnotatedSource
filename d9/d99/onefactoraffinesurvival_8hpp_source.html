34 While deriving from the hazard rate class the HRTS refers only to the
35 deterministic part of the model. The probabilities depend on this
36 component and the stochastic part and are rewritten here.

80 /*!

84 \param dFwd Time of the forward survival calculation and HR

86 \param dTgt Target time of survival probability.
87 \param yVal Realized value of the HR at time dFwd.
88 \param extrapolate Allow curve extrapolation.
89 \return Survival probability.

91 \todo turn into a protected method to be called by
92 defaults and survivals
93 */
94 /*
95 Note: curve extrapolation has a different meaning on different curve

100 */

148 inline Probability
149 OneFactorAffineSurvivalStructure::survivalProbabilityImpl(
150 Time t) const
151 {
152 Real initValHR =
153 model_->dynamics()->shortRate(0.,
154 model_->dynamics()->process()->x0());
155
156 return model_->discountBond(0., t, initValHR);
157 }

165 inline Real
166 OneFactorAffineSurvivalStructure::defaultDensityImpl(Time t) const {
167 Real initValHR =
168 model_->dynamics()->shortRate(0.,
169 model_->dynamics()->process()->x0());;

171 return hazardRateImpl(t)*survivalProbabilityImpl(t) /
